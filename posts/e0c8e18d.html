生成的文件默认存放路径：

  • Windows：C:\Users\你的用户名\.ssh\

其中 id_rsa（或 id_ed25519）是私钥，id_rsa.pub（或 id_ed25519.pub）是公钥。

⚠️ 私钥请妥善保管，切勿泄露给他人。

GitHub 配置

将公钥文件（.pub）中的内容复制，然后登录 GitHub：

  1. 点击右上角头像，进入 Settings
  2. 左侧菜单选择 SSH and GPG keys
  3. 点击 New SSH key
  4. Title 填写一个便于识别的名称（如设备名称）
  5. Key 粘贴公钥内容，点击 Add SSH key 保存
